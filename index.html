IfcOpenShell

The open source IFC toolkit
and geometry engine

IfcOpenShell helps you develop digital platforms for the built environment.
Read, write, and modify Building Information Models using IFC,
a diverse digital language from design to construction and beyond.

Download v0.7.0

A library you can rely on

Developed by a community of hundreds of developers and trusted to deliver many AEC technologies that power our industry.

Any technology stack

Build with Windows, Mac, or Linux, in C++ or Python. Deploy headless server tools, or author IFC natively with a rich graphical interface.

All schemas and formats

Support IFC2X3, IFC4, IFC4.3, or load your own at run-time. Read and write IFC-SPF, IFCJSON, IFCXML, and IFCHDF5.

Native IFC authoring API

High level API for hundreds of tasks. Perform complex authoring like copying objects, cost calculation, or 4D simulation with one line of code.

Robust geometry

Convert parametric geometry into explicit geometry for any CAD system from booleans to complex sweeps.

2D drawings

Generate and annotate 2D drawings from 3D geometry with ease. Preserve drawing semantics and link model data to and from drawing symbols.

Ecosystem of tools

Clash detection, model comparison, and conversion to over 10 other formats. Integrate with technologies like IDS, BCF, and bSDD, and more.

Easy to use

Extensive documentation, user guides, academic courses, and a vibrant user community to help you begin your journey.

Let's get started 

# Load and edit a wall name
import ifcopenshell

model = ifcopenshell.open('model.ifc')
walls = model.by_type('IfcWall')
walls[0].Name = 'My wall'

model.write('updated-model.ifc')

# Properties, type, and location of a door
import ifcopenshell.util.element as util

door = model.by_type("IfcDoor")[0]

prop_sets = util.get_psets(door)
door_type = util.get_type(door)
location = util.get_container(door)

# Create a wall on the site
from ifcopenshell.api import run

site = run("root.create_entity", model,
    ifc_class="IfcSite")
wall = run("root.create_entity", model,
    ifc_class="IfcWall")

run("spatial.assign_container", model,
    relating_structure=site, product=wall)

# Convert a model to different formats
$ IfcConvert model.ifc model.glb

# Multicore processing
$ IfcConvert model.ifc model.dae

# Filters and operations
$ IfcConvert --model-offset "10000;10000;0" \
    model.ifc model.dae

# Check BIM requirements
$ ifctester specs.ids model.ifc

# Produce reports
$ ifctester -r Html -o out.html test.ids bldg.ifc

# Compare models
$ ifcdiff old.ifc new.ifc

# Schedule objects in a spreadsheet
$ ifccsv -i input.ifc -c out.csv -q ".IfcWall"

# Clash detection
$ ifcclash -o results.json clashsets.json
